This should be recognized as an HTML block:

The quick brown fox jumps over the lazy dog.

And this too:

Hello world!

And this (using spaces):

Section heading

Subhead

And so on:

But these should still be code blocks:

<section>
    <p>The quick brown fox jumps over the lazy dog.</p>
</section>

Same with spaces:

<header>
    <hgroup>
        <h1>Section heading</h1>
        <h2>Subhead</h2>
    </hgroup>
</header>
